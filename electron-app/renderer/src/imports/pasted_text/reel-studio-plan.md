# Reel Studio - 디자인 기획서

## 1. 제품 개요

### 1.1 서비스명
**Reel Studio** - AI 기반 숏폼 영상 자동 생성 플랫폼

### 1.2 핵심 가치
- 대본만 입력하면 AI가 이미지 + 음성 + 영상을 자동 생성
- 단계별 확인으로 품질 컨트롤 가능
- 개별 재생성으로 원하는 결과물 확보

### 1.3 타겟 유저
- 인스타그램/틱톡 숏폼 크리에이터
- 교육/정보 콘텐츠 제작자
- 마케팅 담당자

---

## 2. 유저 플로우

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  [시작] ──▶ [대본 입력] ──▶ [AI 분석] ──▶ [이미지 생성]        │
│                                              │                  │
│                                              ▼                  │
│  [완료] ◀── [영상 생성] ◀── [확정] ◀── [이미지 확인/재생성]    │
│     │                                                           │
│     ▼                                                           │
│  [다운로드/공유]                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.1 상세 플로우

| Step | 화면 | 액션 | 결과 |
|------|------|------|------|
| 1 | 대본 입력 | 텍스트 붙여넣기 → "분석하기" 클릭 | AI가 세그먼트 분석 |
| 2 | 분석 중 | 로딩 애니메이션 | 세그먼트 목록 생성 |
| 3 | 이미지 생성 | 자동 병렬 생성 | 각 세그먼트별 이미지 |
| 4 | 이미지 확인 | 프리뷰 확인 → 재생성/확정 | 만족스러운 이미지 확보 |
| 5 | 영상 생성 | "영상 생성" 클릭 → 자동 생성 | 음성+영상 생성 |
| 6 | 완료 | 영상 프리뷰 → 다운로드 | 최종 결과물 |

## 3. 페이지별 상세 기획

```
┌─────────────────────────────────────────────────────────────────┐
│  🎬 Reel Studio                              [히스토리] [설정]  │
│  대본 → 이미지 → 영상, AI가 자동으로                           │
```

**구성 요소:**
- 로고 + 서비스명
- 서브 태그라인
- 히스토리 버튼 (이전 작업 목록)
